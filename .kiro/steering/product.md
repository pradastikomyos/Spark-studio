# Product Overview

Spark Photo Studio is a premium photography studio booking and e-commerce platform. The application enables customers to book studio sessions, purchase tickets for events/workshops, and buy photography-related products.

## Core Features

- **Studio Booking System**: Date-based ticket booking with availability management
- **Event Management**: Workshops, exhibitions, masterclasses, and seminars
- **E-commerce**: Product catalog with shopping cart functionality
- **Payment Integration**: Midtrans payment gateway with webhook support
- **Admin Dashboard**: Ticket management, inventory control, stage/QR management, analytics
- **Authentication**: User accounts with role-based access (admin/customer)
- **QR Code System**: Stage access control with QR scanning capabilities

## Business Model: Dufan-Style Multi-Stage Access

The system uses a theme park model where one ticket grants access to multiple stages (photo studio rooms). Key aspects:

- **Single Ticket, Multiple Stages**: Customers purchase one ticket that allows entry to various studio stages
- **Free QR Scanning**: Users scan QR codes at each stage entrance (no additional charge)
- **Traffic Analytics**: QR scans track which stages are popular and which need promotion
- **Stage Analytics Dashboard**: Admin can view traffic patterns, peak times, and stage popularity metrics

This model enables data-driven decisions about stage operations, staffing, and marketing focus.

## User Roles

- **Customers**: Browse, book sessions, purchase tickets/products, manage their bookings
- **Admins**: Full access to dashboard, ticket management, inventory, analytics, and QR generation

## Design Philosophy

Premium, sophisticated aesthetic with dark mode as the default. Emphasis on elegance, high contrast, smooth transitions, and professional photography studio branding.
